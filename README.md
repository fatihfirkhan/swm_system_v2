This is my Final Year Project (FYP) for Bachelor of Information Technology at Universiti Tun Hussein Onn Malaysia.

**Project Summary:**  
A web-based Waste Management System that allows staff to schedule, track, and update waste collection status. Residents can view schedules and report missed collections with photo evidence.

**Key Technologies:**  
- Laravel (PHP), MySQL  
- HTML, CSS, JavaScript  
- User roles, status tracking, reporting features

**Features:**  
✔ Schedule management  
✔ Collection status updates  
✔ Resident report submission with photos  
✔ Simple dashboard and user interfaces
